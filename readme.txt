This plugin adds some extended functionality for the s2member plugin. These extensions are unofficial and unsupported.

== Installation ==

1. Upload the s2-extensions folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
